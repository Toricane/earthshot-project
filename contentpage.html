Introduction to Linear Algebra

1. Vector Basics

A vector is a mathematical object that has both magnitude and direction. In contrast, a scalar has only magnitude.

Vectors can be represented in multiple ways:

  • Geometrically as an arrow in a coordinate system
  • Algebraically as an ordered list of numbers
Example: The vector v = (3, 4) has a magnitude of |v| = √(3² + 4²) = √25 = 5 and points in the direction given by arctan(4/3) ≈ 53.1° from the positive x-axis.
2. Vector Addition

Vectors can be added by combining their components:

v + w = (v₁ + w₁, v₂ + w₂, ..., vₙ + wₙ)

Geometrically, vector addition follows the parallelogram law or the tip-to-tail method, where the sum of two vectors forms the diagonal of the parallelogram created by the two vectors.

Example: If v = (2, 3) and w = (4, 1), then v + w = (2+4, 3+1) = (6, 4)
3. Scalar Multiplication

A vector can be multiplied by a scalar (a number) to change its magnitude:

c · v = (c · v₁, c · v₂, ..., c · vₙ)

If c is positive, the direction remains the same but the magnitude changes by a factor of |c|. If c is negative, the direction is reversed and the magnitude changes by a factor of |c|.

Example: If v = (3, -2) and c = 2, then c · v = 2 · (3, -2) = (6, -4)
The magnitude increases from |v| = √(3² + (-2)²) = √13 to |c·v| = √(6² + (-4)²) = √52 = 2√13
4. Vector Spaces

A vector space is a collection of vectors that is closed under addition and scalar multiplication. This means that when you add two vectors from the space or multiply a vector by a scalar, the result is still in the space.

A vector space must satisfy the following axioms:

  • Closed under addition: u + v is in the space for any vectors u and v in the space.
  • Closed under scalar multiplication: c · v is in the space for any scalar c and any vector v in the space.
  • Commutativity of addition: u + v = v + u for all vectors u and v in the space.
  • Associativity of addition: (u + v) + w = u + (v + w) for all vectors u, v, and w in the space.
  • Existence of zero vector: There exists a vector 0 such that v + 0 = v for all vectors v in the space.
  • Existence of additive inverse: For each vector v, there exists a vector -v such that v + (-v) = 0.
Example: ℝ² (the plane) is a vector space where the vectors are ordered pairs of real numbers.
5. Linear Independence

A set of vectors is linearly independent if none of them can be expressed as a linear combination of the others. Mathematically, vectors v₁, v₂, ..., vₙ are linearly independent if the equation:

c₁v₁ + c₂v₂ + ... + cₙvₙ = 0

has only the trivial solution c₁ = c₂ = ... = cₙ = 0.

If a set of vectors is not linearly independent, then it is linearly dependent, which means at least one of the vectors can be expressed as a linear combination of the others.

Example: The vectors v₁ = (1, 0) and v₂ = (0, 1) are linearly independent in ℝ². However, if we add v₃ = (1, 1), the set {v₁, v₂, v₃} is linearly dependent because v₃ = v₁ + v₂.
